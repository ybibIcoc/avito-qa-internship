# Тест-кейсы: Платформа модерации объявлений

**Окружение:** `https://cerulean-praline-8e5aa6.netlify.app`
**Особенность:** Данные на стенде динамические. Тест-кейсы не привязываются к конкретному количеству элементов, а проверяют логику отображения и изменения состояний.

---

## 1. Десктопная версия (Разрешение ПК)

### TC-01: Проверка работы фильтра "Диапазон цен"
**Предусловие:** Открыта главная страница (`/`). Дождаться окончания загрузки данных.
**Шаги:**
1. В боковом меню фильтров ввести в поле "От" значение `10000`.
2. Нажать `Enter` (или убрать фокус с поля).
3. Ввести в поле "До" значение `50000` и нажать `Enter`.
4. Дождаться обновления списка объявлений.
5. Проверить цену на всех отображаемых карточках.
**Ожидаемый результат:** В выдаче присутствуют только те карточки, цена которых находится в диапазоне `10 000 ₽ <= Цена <= 50 000 ₽`.

### TC-02: Проверка сортировки "По цене"
**Предусловие:** Открыта главная страница (`/`).
**Шаги:**
1. В верхнем меню в выпадающем списке "Сортировать по" выбрать значение "Цене".
2. В соседнем выпадающем списке "Порядок" выбрать значение "По возрастанию".
3. Дождаться обновления списка объявлений.
4. Проверить последовательность цен в карточках сверху вниз.
**Ожидаемый результат:** Цены в карточках отсортированы строго по возрастанию (каждая последующая цена больше или равна предыдущей).

### TC-03: Проверка работы фильтра "Категория"
**Предусловие:** Открыта главная страница (`/`).
**Шаги:**
1. В боковом меню в выпадающем списке "Категория" выбрать значение "Детское".
2. Дождаться обновления списка объявлений.
3. Проверить заголовки всех отображаемых карточек.
**Ожидаемый результат:** В выдаче присутствуют только карточки, в заголовке которых содержится слово "Детское".

### TC-04: Проверка работы тогла "Только срочные"
**Предусловие:** Открыта главная страница (`/`).
**Шаги:**
1. Активировать переключатель (тогл) "🔥 Только срочные" в верхней панели.
2. Дождаться обновления списка объявлений.
3. Проверить наличие соответствующих бейджей на карточках.
**Ожидаемый результат:** На всех отображаемых карточках присутствует визуальный бейдж "⚡ Срочно". Карточки без бейджа скрыты.

### TC-05: Управление таймером обновления статистики
**Предусловие:** Совершить переход с главной страницы на страницу "Статистика" (`/stats`).
**Шаги:**
1. Нажать кнопку "Отключить автообновление".
   * **ОР 1:** Таймер скрывается, появляется надпись "Автообновление выключено". Значения статистики зафиксированы.
2. Нажать кнопку "Включить автообновление".
   * **ОР 2:** Появляется текст "Обновление через:", таймер возобновляет отсчет секунд вниз.
3. Нажать кнопку "Обновить сейчас".
   * **ОР 3:** Значение таймера мгновенно сбрасывается к исходному значению (например, 5:00) и начинается новый отсчет.

## 2. Мобильная версия (Эмуляция смартфона)

### TC-06: Переключение между тёмной и светлой темой
**Предусловие:** Открыта главная страница в мобильном разрешении (например, Pixel 5).
**Шаги:**
1. Проверить текущее значение атрибута `data-theme` в теге `<html>`.
2. Нажать на кнопку переключения темы (иконка в верхнем правом углу).
3. Проверить изменение атрибута `data-theme`.
**Ожидаемый результат:** Атрибут `data-theme` меняется на противоположный (`light` на `dark` или наоборот). Происходит визуальная смена цветовой схемы интерфейса.
