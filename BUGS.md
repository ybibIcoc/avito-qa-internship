# Найденные дефекты (Bug Reports)

В ходе прогона UI-автотестов были выявлены 3 функциональных дефекта логики приложения и 1 дефект инфраструктуры стенда. 

Автотесты **TC-01**, **TC-04** и **TC-05** написаны корректно и ожидаемо **падают**, фиксируя данные баги (согласно требованиям задания).

---

## BUG-01: Не работает фильтрация объявлений по "Диапазону цен"
*   **Серьезность:** P1 (High)
*   **Окружение:** Desktop / Mobile 

**Шаги воспроизведения:**
1. На главной странице ввести в поле "От" значение `10000`.
2. Ввести в поле "До" значение `50000`.
3. Снять фокус с полей (или нажать Enter), дождаться завершения загрузки.

**Фактический результат:**
Фильтрация не применяется. В выдаче продолжают отображаться карточки с ценой, выходящей за рамки диапазона (например, > 80 000 ₽). Общее количество объявлений не меняется.
**Ожидаемый результат:**
Список обновляется. В выдаче отображаются только объявления с ценой от 10 000 ₽ до 50 000 ₽.

---

## BUG-02: Не работает тогл "Только срочные"
*   **Серьезность:** P1 (High)
*   **Окружение:** Desktop / Mobile 

**Шаги воспроизведения:**
1. На главной странице активировать переключатель "🔥 Только срочные".
2. Дождаться обновления выдачи.

**Фактический результат:**
Тогл визуально переключается, однако список объявлений не фильтруется. В выдаче по-прежнему присутствуют обычные объявления без бейджа "⚡ Срочно".
**Ожидаемый результат:**
В списке остаются исключительно объявления, помеченные бейджем "⚡ Срочно".

---

## BUG-03: Не работает кнопка "Включить автообновление" (Play) на странице статистики
*   **Серьезность:** P2 (Medium)
*   **Окружение:** Desktop / Mobile (`/stats`)

**Шаги воспроизведения:**
1. На странице статистики в блоке таймера нажать кнопку "Отключить автообновление" (Пауза).
2. Убедиться, что появилась надпись "Автообновление выключено".
3. Нажать на кнопку "Включить автообновление" (Play ▶️).

**Фактический результат:**
Интерфейс не реагирует на нажатие. Таймер не запускается, надпись "Автообновление выключено" остается на экране.
**Ожидаемый результат:**
Таймер возобновляет работу, появляется текст "Обновление через: [время]", начинается обратный отсчет.

## BUG-04: Ошибка 404 при прямом переходе (Direct Navigation) по URL
*   **Серьезность:** P3 (Low)
*   **Причина:** Отсутствие файла `_redirects` для SPA на сервере Netlify.

**Шаги воспроизведения:**
1. Ввести в адресную строку браузера прямой URL: `https://cerulean-praline-8e5aa6.netlify.app/stats`
2. Нажать Enter.

**Фактический результат:**
Открывается системная страница Netlify "Page Not Found" (Ошибка 404).
**Ожидаемый результат:**
Приложение успешно маршрутизирует пользователя на страницу статистики.
