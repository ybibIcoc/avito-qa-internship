## 🚀 Инструкция по локальному запуску

Проект настроен так, чтобы запускаться стандартными командами без дополнительных сложных настроек окружения.

### 1. Клонирование и установка зависимостей
Убедитесь, что у вас установлен Node.js (v16+). Откройте терминал и выполните команды:

```bash
git clone <ссылка-на-ваш-репозиторий>
cd <название-папки-репозитория>
npm install
npx playwright install --with-deps
```

### 2. Запуск тестов

**Запуск всех тестов в фоновом режиме:**
```bash
npx playwright test
```

**Запуск с визуальным отображением браузера:**
```bash
npx playwright test --project="Desktop Firefox" --headed  
```

**Запуск только мобильного тестирования (TC-06):**
```bash
npx playwright test mobile.spec.ts --project="Mobile Android (Pixel 5)" --headed
```

### 3. Просмотр отчета
Playwright автоматически генерирует подробный HTML-отчет (с логами и деталями падения тестов из-за найденных багов). Для его просмотра выполните:
```bash
npx playwright show-report
```
